# Randget - Budget App

Randget is a simple Android budget tracking app that helps users manage their expenses and stay within financial goals.

---

## Features

- User Registration & Login
- Add Expense Entries
- Attach Images to Expenses
- Categorize Expenses (Food, Transport, etc.)
- Set Minimum & Maximum Budget Goals
- View Expenses by Date Range
- View Total Spending Per Category
- Dashboard with Budget Status

---

## Built With

- Kotlin
- Android Studio
- Room Database (Offline Storage)
- RecyclerView

---


##  How to Run

1. Clone the repository
2. Open in Android Studio
3. Sync Gradle
4. Run on Emulator or Physical Device
